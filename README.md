# Hypnos - Screen Time Management 

This project is a simple **Node.js Express** web application following the **Model–View–Controller (MVC)** architecture. It uses **Handlebars (hbs)** as the view engine and supports static assets via a public directory.

Hypnos is a web application designed to help users track, manage, and reduce their screen time through features like activity timers, goal setting, mood tracking, and insightful analytics.

---
## Installation and Setup

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Server
```bash
node app.js
```

You should see this message in the terminal:
```
app listening at port 3000
```

### 4. Open the App
Visit your app in the browser at:  
[http://localhost:3000](http://localhost:3000)
